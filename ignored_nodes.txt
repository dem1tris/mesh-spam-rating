# Один ID ноды в строке, в том же формате, что и поле from_node_id: с ведущим '!' (например, !0b86cb0c).
# Строки, начинающиеся с '#', считаются комментариями.
#
# Здесь должен быть ID ноды, с которой ведётся захват пакетов (приёмник / «базовая» нода).
# Эта нода сохраняет в дамп много пакетов, которые не уходят в сеть.
# Ноды из этого списка будут полностью игнорироваться при построении статистики.
#

# Пример
#!qqqq9876 # удалите символ '#' из начала строки, замените id на id своей ноды
